cartuccia di generazione del grafico di un'orchestrazione

La cartuccia it/csi/mddtools/servicegen/workflow/csi14.mwe permette la generazione di un report grafico che descrive l'orchestrazione in un formato simile ai flow-chart. La cartuccia prevede i seguenti parametri:

parametrodescrizioneesempioobbl/opz
model percorso del file del modello servicegen, a partire dalla root del workspace eclipse myprod.mdd/src/model/myprod/mycomp/myservice.servicegen obbl
targetDir percorso della cartella in cui devono essere generati i file di report (il percorso è relativo alla directory root del progetto) docs/reports obbl
graphvizBin Percorso completo della directory in cui si trovano i files eseguibuli di graphviz. c:/programmi/graphviz/bin/ obbl

Esempio: 

<cartridge file="it/csi/mddtools/servicegen/workflow/orchestrationReport.mwe"
  model="myprod.mdd/src/model/myprod/mycomp/myservice_orch1.svcorch" 
  targetDir="docs/reports"
  graphvizBin="c:/programmi/graphviz/bin/"
/>

Il risultato dell'esecuzione è rappresentato da:

  • un file in formato .dot che deve essere dato in pasto al tool graphviz(*) per generare l'immagine del report
  • due batch (uno per windows e uno per unix) che devono essere eseguiti per completare il lavoro e produrre la gif che rappresenta il report

(*) il tool graphviz può essere scaricato all'indirizzo http://www.graphviz.org/

Class "typedef :: PrimitiveType"

Defined in package : typedef

Tipo base customizzabile. Permette di definire differenti Typeset. In particolare è possibile definire anche un insieme di language binding che permettono di indicare il tipo language specific a cui deve essere associato il tipo modellato.
